## Installation

 * `apt-get install graphviz qtcreator`
 * `./download_other_githubs.sh`

## Building

Either load the `.pro` file from Qt Creator, or use `qmake` to build:

 * `qmake my.pro`: creates a makefile
 * `make`: build that makefile
